# 🚀 Quick Start Guide - Activity Feed & Profile Features

## ✅ What's New

You now have a complete activity tracking and profile management system!

---

## 🎯 How to Use

### **1. Access Your Profile**
```
Click your avatar/name (top right) → "Edit Profile"
```
- Upload an avatar (or paste URL)
- Update your name and email
- Add a bio (optional)
- Click "Save Changes"

### **2. View Activity Feed**
```
Click your avatar/name (top right) → "Activity Feed"
```
- See activities from ALL your documents
- Filter by type (Proposals, Votes, Comments)
- Filter by specific document
- View statistics dashboard
- Activities auto-refresh every 30 seconds

### **3. Logout**
```
Click your avatar/name (top right) → "Logout"
```

## 🎨 Activity Feed Features

### **Statistics Dashboard**
See at-a-glance:
- Total activities across all documents
- Number of proposals created
- Number of votes cast
- Number of comments added
- Number of proposals accepted

### **Smart Filtering**
- **By Type**: View only proposals, votes, comments, or acceptances
- **By Document**: Focus on a specific document
- **Combined**: Filter by both type and document

### **Date Grouping**
Activities organized by:
- Today
- Yesterday
- Older dates (grouped by date)

### **Activity Details**
Each activity shows:
- User avatar and name
- What they did (proposed, voted, commented)
- Which document and section
- When it happened
- Color-coded icon

## 🎬 Quick Demo

### **Update Your Profile:**
1. Start the app: `npm run dev:full`
2. Login with any demo account
3. Click your avatar → "Edit Profile"
4. Paste an image URL (or upload)
5. Add a bio: "Product Manager at Acme Corp"
6. Click "Save Changes"
7. Your avatar appears everywhere!

### **View Activity Feed:**
1. From anywhere, click your avatar
2. Select "Activity Feed"
3. See the statistics dashboard
4. Try filtering: Select "Proposals" from dropdown
5. Try document filter: Select specific document
6. Click "Back" to return

### **Make Activity Appear:**
1. Open a document
2. Suggest a change to any paragraph
3. Vote on a proposal
4. Add a comment
5. Open Activity Feed
6. See your activities listed!
