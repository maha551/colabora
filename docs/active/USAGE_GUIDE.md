# 📖 Activity Feed & Profile Management - Usage Guide

## Quick Start Guide

### 🎯 Activity Feed

The activity feed shows you what's happening in your document in real-time.

#### **Where to Find It**
Look at the **right sidebar** when viewing any document (desktop only).

#### **What You'll See**

```
┌─────────────────────────────────────┐
│ 🕐 Recent Activity            [12]  │
├─────────────────────────────────────┤
│ 👤 Bob Smith                   📝   │
│ suggested a change                  │
│ to "Introduction"                   │
│ 2 minutes ago                       │
├─────────────────────────────────────┤
│ 👤 You                         ✅   │
│ proposal was accepted               │
│ in "Voting Process"                 │
│ 5 minutes ago                       │
├─────────────────────────────────────┤
│ 👤 Alice Johnson               👍   │
│ approved a proposal                 │
│ in "Making Changes"                 │
│ 10 minutes ago                      │
└─────────────────────────────────────┘
```

#### **Activity Types**

| Icon | Color | Activity | What it means |
|------|-------|----------|--------------|
| 📝 | Blue | Proposal Created | Someone suggested a change |
| ✅ | Green | Proposal Accepted | A proposal got 75% approval |
| 👍 | Green | Vote PRO | Someone approved a proposal |
| ➖ | Gray | Vote NEUTRAL | Someone voted neutral |
| 👎 | Red | Vote CONTRA | Someone rejected a proposal |
| 💬 | Purple | Comment Added | Someone left a comment |

#### **Features**
- ✨ **Auto-refresh**: Updates every 30 seconds automatically
- 🔍 **Context**: Shows which section was affected
- 👤 **User info**: See who did what
- ⏰ **Smart times**: "2m ago" instead of full timestamps
- 📜 **Scrollable**: See up to 50 recent activities

### 👤 Profile Management

Customize your profile so others know who you are!

#### **How to Edit Your Profile**

**Step 1: Open Profile Menu**
Click on your **avatar/name** in the top right corner.

```
┌─────────────────────────────────────┐
│                 👤 Alice Johnson  ▼ │ ← Click here
└─────────────────────────────────────┘
```

**Step 2: Select "Edit Profile"**
A dropdown menu appears. Click **"Edit Profile"**.

```
┌──────────────────────┐
│ 👤 Edit Profile      │ ← Click here
├──────────────────────┤
│ 🚪 Logout            │
└──────────────────────┘
```

**Step 3: Edit Your Information**

A dialog opens with your current information:

```
┌───────────────────────────────────────┐
│ Edit Profile                    [×]   │
├───────────────────────────────────────┤
│                                       │
│         ┌─────────┐                   │
│         │   AJ    │ 📷 ← Click camera │
│         └─────────┘      to upload    │
│                                       │
│ Avatar URL (Optional)                 │
│ ┌───────────────────────────────────┐ │
│ │ https://example.com/avatar.jpg    │ │
│ └───────────────────────────────────┘ │
│ Or paste a direct image link         │
│                                       │
│ Name *                                │
│ ┌───────────────────────────────────┐ │
│ │ Alice Johnson                     │ │
│ └───────────────────────────────────┘ │
│                                       │
│ Email *                               │
│ ┌───────────────────────────────────┐ │
│ │ alice@example.com                 │ │
│ └───────────────────────────────────┘ │
│                                       │
│ Bio (Optional)                        │
│ ┌───────────────────────────────────┐ │
│ │ I love collaborative writing!     │ │
│ │                                   │ │
│ └───────────────────────────────────┘ │
│                           35/200      │
│                                       │
│         [Cancel]  [Save Changes]      │
└───────────────────────────────────────┘
```

#### **Field Guide**

1. **Avatar** (3 ways to add one):
   - **Option A**: Click the 📷 camera icon to upload a file
     - Supports: JPG, PNG, GIF
     - Max size: 5MB
   - **Option B**: Paste a URL to an image
   - **Option C**: Leave blank for default initials

2. **Name*** (required)
   - Your display name
   - Shown to all collaborators
   - Used for avatar initials if no image

3. **Email*** (required)
   - Must be unique
   - Valid email format required
   - Used for account identification

4. **Bio** (optional)
   - Tell others about yourself
   - Max 200 characters
   - Shows character counter

**Step 4: Save Changes**
Click **"Save Changes"** when done. Your profile updates everywhere instantly!

## 💡 Pro Tips

### Activity Feed Tips

1. **Stay Informed**: Check the feed regularly to see what's new
2. **Context Clues**: Activity shows section names so you know where to look
3. **Your Activities**: Items from "You" are highlighted in blue
4. **Timestamps**: Hover to see exact time (in the future)
5. **Empty Feed**: If you see "No recent activity", be the first to make a change!

### Profile Tips

1. **Good Avatars**: Use a clear, professional photo or fun illustration
2. **Image Size**: Keep images under 5MB for fast loading
3. **Bio Ideas**: 
   - "Product designer passionate about UX"
   - "Technical writer | Documentation enthusiast"
   - "Engineering manager at Acme Corp"
4. **Update Often**: Change your avatar seasonally or for special occasions

## 🔒 Privacy & Security

### What's Public
- ✅ Your name (visible to all document collaborators)
- ✅ Your avatar (visible to all document collaborators)
- ✅ Your bio (visible to all document collaborators)
- ✅ Your activities in documents you collaborate on

### What's Private
- 🔒 Your email (only you can see it)
- 🔒 Your activities in documents others can't access
- 🔒 Your profile editing actions

### Security Features
- 🛡️ Authentication required for all actions
- 🛡️ Can only see activities in documents you have access to
- 🛡️ Email must be unique (prevents impersonation)
- 🛡️ Profile updates are validated server-side

## 🆘 Troubleshooting

### Activity Feed Issues

**Problem**: "No activities showing"
- ✅ **Solution**: This is normal for new documents. Start collaborating!

**Problem**: "Activities not updating"
- ✅ **Solution**: They auto-refresh every 30 seconds. Wait a moment or refresh the page.

**Problem**: "Can't see activity feed"
- ✅ **Solution**: Activity feed only shows on desktop screens. Use a larger screen or check activities after actions on mobile.

### Profile Issues

**Problem**: "Email already in use"
- ✅ **Solution**: Someone else is using that email. Try a different one.

**Problem**: "Image too large"
- ✅ **Solution**: Use an image under 5MB. Resize it using an image editor.

**Problem**: "Can't upload avatar"
- ✅ **Solution**: Make sure it's an image file (JPG, PNG, GIF). Try using an avatar URL instead.

**Problem**: "Changes not saving"
- ✅ **Solution**: Check that name and email are filled in. Look for error messages below fields.
