# Colabora Admin Setup Guide

See [docs/active/ADMIN_SETUP.md](docs/active/ADMIN_SETUP.md) for the full admin setup and API reference.

Quick start:

```bash
export ADMIN_SETUP_EMAIL="admin@yourdomain.com"
export ADMIN_SETUP_PASSWORD="your-secure-password-min-12-chars"
npm run db:migrate
npm run setup-admin
```
