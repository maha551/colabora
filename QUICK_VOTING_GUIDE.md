# 🗳️ Quick Voting Guide - Activity Feed

## ⚡ How to Vote on Proposals Quickly

### **Step 1: Open Activity Feed**
Click your avatar/name → "Activity Feed"

---

### **Step 2: See Pending Proposals**

If there are proposals needing your vote, you'll see an **orange box** at the top:

```
┌──────────────────────────────────────────────────┐
│ ⚠️  Pending Your Vote                      [3]   │
│ These proposals need your vote. Review below.    │
├──────────────────────────────────────────────────┤
│ [Scrollable list of proposals]                   │
└──────────────────────────────────────────────────┘
```

---

### **Step 3: Review Each Proposal**

Each proposal shows:

```
┌────────────────────────────────────────────────┐
│ 👤 Bob Smith               [Body Change]      │
│ Sample Document • Introduction • 2 hours ago   │
│                                                │
│ ┌──────────────────────────────────────────┐  │
│ │ CURRENT TEXT:                            │  │
│ │ This is the first paragraph of our       │  │
│ │ collaborative document.                  │  │
│ │                                          │  │
│ │ PROPOSED TEXT:                           │  │
│ │ This is the initial paragraph of our    │  │
│ │ collaborative document.                  │  │
│ │ ^^^^^^^^                                 │  │
│ │ (differences highlighted)                │  │
│ └──────────────────────────────────────────┘  │
│                                                │
│ Current votes: 👍 2  ➖ 1  👎 0               │
│                                                │
│ [Approve] [Neutral] [Reject]                  │
└────────────────────────────────────────────────┘
```

---

### **Step 4: Vote**

Click one of three buttons:

- **[Approve]** - Green button - You like the change
- **[Neutral]** - Gray button - You're okay either way
- **[Reject]** - Red button - You don't like the change

---

### **Step 5: Done!**

After voting:
1. ✅ Success toast appears: "Voted pro"
2. 🗑️ Proposal disappears from list
3. 📊 Counter updates [3] → [2]
4. 🔄 Activity feed refreshes

---

## 💡 Pro Tips

### **Tip 1: Review Context**
Always check:
- ✅ Who made the proposal (trust level)
- ✅ Which document it's for (importance)
- ✅ What section (context)
- ✅ How others voted (consensus)

### **Tip 2: Use the Diff**
The diff view highlights:
- 🟢 **Green** = Added text
- 🔴 **Red** = Removed text
- ⚪ **Gray** = Unchanged text

### **Tip 3: Vote in Batches**
- Open Activity Feed once per day
- Vote on all proposals at once
- Scroll through the list
- Much faster than document-by-document

### **Tip 4: Check Vote Counts**
Before voting, see what others think:
- **👍 High PRO votes**: Probably good to approve
- **👎 High CONTRA votes**: Consider rejecting
- **Mix of votes**: Read carefully before deciding

## 🔄 What Happens After You Vote?

### **Immediately:**
1. Your vote is recorded
2. Proposal disappears from your pending list
3. Your vote appears in activity feed
4. Proposal's vote count updates

### **If Proposal Reaches 75%:**
1. Proposal gets accepted automatically
2. Text updates in document
3. "Proposal Accepted" activity appears
4. Document shows new version

### **If Proposal is Rejected:**
1. Stays in proposals list (not accepted)
2. Author can see rejection
3. Author may create new proposal with changes

## ❓ FAQ

### **Q: What if I vote wrong?**
**A**: Currently, votes are final. Vote carefully! (Future: vote changing may be added)

### **Q: Can I see who voted what?**
**A**: You see vote counts, but not individual voter names (by design for privacy).

### **Q: What if I'm unsure?**
**A**: Vote "Neutral"! It's better than not voting at all.

### **Q: Do I have to vote on everything?**
**A**: Technically no, but it helps the team reach consensus faster.

### **Q: What's the 75% threshold?**
**A**: Proposals need 75% PRO votes to be accepted. If 4 collaborators, need 3 PRO votes.

### **Q: Can I comment before voting?**
**A**: Yes! Click the proposal to open the full view, add comments, then vote.
